## Стъпка 1: Конвертиране на Markdown към Word

### Вариант A: Използване на Pandoc (Препоръчително)

1. **Инсталирайте Pandoc:**
   - Изтеглете от: https://pandoc.org/installing.html
   - Инсталирайте на вашия компютър

2. **Конвертирайте документа:**
   Отворете Command Prompt или PowerShell в папката на проекта и изпълнете:
   ```powershell
   pandoc ДОКУМЕНТАЦИЯ_ПРОЕКТ.md -o ДОКУМЕНТАЦИЯ_ПРОЕКТ.docx --reference-doc=reference.docx
   ```

3. **Ако нямате референционен файл, използвайте:**
   ```powershell
   pandoc ДОКУМЕНТАЦИЯ_ПРОЕКТ.md -o ДОКУМЕНТАЦИЯ_ПРОЕКТ.docx -s
   ```

### Вариант B: Използване на онлайн конвертор

1. Отворете: https://www.markdowntoword.com/
2. Качете файла `ДОКУМЕНТАЦИЯ_ПРОЕКТ.md`
3. Изтеглете генерирания Word документ

### Вариант B: Използване на Visual Studio Code

1. Инсталирайте разширението "Markdown PDF"
2. Отворете `ДОКУМЕНТАЦИЯ_ПРОЕКТ.md`
3. Натиснете `Ctrl+Shift+P`
4. Изберете "Markdown PDF: Export (docx)"

### Вариант C: Ръчно копиране

1. Отворете `ДОКУМЕНТАЦИЯ_ПРОЕКТ.md` в Word
2. Word автоматично ще го форматира
3. Ръчно форматирайте заглавията и стиловете

## Стъпка 3: Добавяне на скрийншоти в Word

### Инструкции:

1. **Отворете Word документа** (след конвертиране)

2. **Намерете местата с `[СНИМКА: ...]`**

3. **Вмъкнете скрийншотите:**
   - Поставете курсора на мястото на `[СНИМКА: ...]`
   - Изберете Insert → Pictures → This Device
   - Изберете съответния скрийншот
   - Изтрийте текста `[СНИМКА: ...]`

4. **Форматиране на скрийншотите:**
   - Под всеки скрийншот добавете подпис:
     - Изберете скрийншота
     - Кликнете с десен бутон → Insert Caption
     - Или просто напишете под него: "Фигура 1: [Описание]"

5. **Настройка на размера:**
   - Скрийншотите трябва да са с подходящ размер
   - Препоръчително: Ширина 15-16 cm
   - Запазете пропорциите

## Стъпка 4: Финален форматиране

### Стилизиране:

1. **Заглавия:**
   - Използвайте стилове Heading 1, Heading 2, Heading 3
   - Форматирайте автоматично чрез Styles панела

2. **Таблици:**
   - Форматирайте всички таблици с граници
   - Използвайте Table Styles

3. **Кодови блокове:**
   - Използвайте моноширинна шрифт (Consolas, Courier New)
   - Добавете фон (светло сиво)

4. **Оглавление:**
   - След форматиране, обновете оглавлението
   - References → Table of Contents → Update Table

5. **Страници:**
   - Добавете номера на страниците
   - Insert → Page Number
