## 🚀 Следващи стъпки:

### Стъпка 1: Конвертиране в Word

**Вариант A - Pandoc (най-добър):**
```powershell
# Инсталирайте Pandoc от: https://pandoc.org/installing.html
# След това:
pandoc ДОКУМЕНТАЦИЯ_ПРОЕКТ.md -o ДОКУМЕНТАЦИЯ_ПРОЕКТ.docx -s
```

**Вариант B - Онлайн:**
- Отворете: https://www.markdowntoword.com/
- Качете `ДОКУМЕНТАЦИЯ_ПРОЕКТ.md`
- Изтеглете Word документа

**Вариант C - Visual Studio Code:**
- Инсталирайте разширението "Markdown PDF"
- Отворете `ДОКУМЕНТАЦИЯ_ПРОЕКТ.md`
- `Ctrl+Shift+P` → "Markdown PDF: Export (docx)"

### Стъпка 2: Вземете скрийншоти

Следвайте **SCREENSHOTS_CHECKLIST.md** за списък с всички 22 скрийншота.

**Бърз списък:**
1. Начална страница
2. Вход / Регистрация
3. Сервитьорски панел
4. Създаване на поръчка
5. Кухненски панел
6. Плащане (в брой и карта)
7. Мениджърски отчети
8. Резервации

### Стъпка 3: Добавете скрийншоти в Word

1. Отворете конвертирания Word документ
2. Намерете местата с `[СНИМКА: ...]`
3. Изтрийте `[СНИМКА: ...]` текста
4. Вмъкнете скрийншота на това място (Insert → Pictures)
5. Добавете подпис под всеки скрийншот

### Стъпка 4: Финален форматиране

- Форматирайте заглавията (Heading 1, 2, 3)
- Обновете оглавлението (References → Table of Contents)
- Добавете номера на страниците
- Проверете, че всичко е наред
